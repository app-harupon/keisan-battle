# ケイサンバトル

計算 or 漢字の読み方を解いて相手を攻撃する、ブラウザで動く対戦ゲーム(試作版)。
単一HTMLファイル(`index.html`)で動作し、ビルド不要・依存ライブラリなし。

## 遊び方

`index.html` をブラウザで開くだけ。スマホでのプレイを想定した設計。

- **ゲーム種別**: 「けいさん」「かんじの読み方」から選択
- **スピードバトル**: 問題を解いて攻撃。相手も自動攻撃してくる。先にHPを0にした方が勝ち
- **サバイバル**: 制限時間内に解き続け、5ミスで終了。スコア(正解数)を伸ばす
- **レベルモード**: Lv.1〜10のステージクリア型。クリアで次ステージが解放

### 特徴的な仕組み

- **攻撃演出**: 解答すると攻撃が約1秒かけて相手側へ飛び、着弾時にダメージ。双方の攻撃が同時に飛んでいると「相殺(クラッシュ)」が発生し、威力の差分だけが届く
- **入力**: 計算モードは画面上テンキー(スマホキーボードを出さない)、漢字モードは端末標準のひらがな入力。物理キーボードにも対応
- **パス**: 現在の問題をノーカウントでスキップ(3秒クールタイム)
- **難易度**: 計算はレベルで桁数が増加(Lv1-2:1桁 〜 Lv10:5桁)、漢字は漢検の級を目安にスライド(Lv1=10級〜8級 … Lv10=2級〜1級)

## 開発

ローカルで開発する場合:

```
# 任意の静的サーバーで配信(例)
npx serve .
```

もしくは `index.html` を直接ブラウザで開いてもOK。

# ケイサンバトル

計算 or 漢字の読み方を解いて相手を攻撃する、ブラウザで動く対戦ゲーム(試作版)。
`index.html`(ゲーム本体)+ `kanji-data.js`(漢字語彙データ、全10レベル・約720語)の2ファイル構成。ビルド不要・依存ライブラリなし。

## 遊び方

`index.html` をブラウザで開くだけ。スマホでのプレイを想定した設計。

- **教科**: 「けいさん」「かんじの読み方」から選択(れきし・かがく・えいご・おんがく・びじゅつは近日公開。今は選ぶと「準備中」通知が出る)
- **スピードバトル**: 問題を解いて攻撃。相手も自動攻撃してくる。先にHPを0にした方が勝ち
- **サバイバル**: 制限時間内に解き続け、5ミスで終了。スコア(正解数)を伸ばす
- **レベルモード**: Lv.1〜10のステージクリア型。クリアで次ステージが解放
- **ちょうどいいバトル**: レベル選択なしで即挑戦。正解が続くと問題の難易度が自動で上がり、間違えるとやさしく戻る。自分に合った難易度を遊びながら見つけられる

### 特徴的な仕組み

- **攻撃演出**: 解答すると攻撃が約1秒かけて相手側へ飛び、着弾時にダメージ。双方の攻撃が同時に飛んでいると「相殺(クラッシュ)」が発生し、威力の差分だけが届く
- **入力**: 計算モードは画面上テンキー(スマホキーボードを出さない)、漢字モードは端末標準のひらがな入力。物理キーボードにも対応
- **パス**: 現在の問題をノーカウントでスキップ(3秒クールタイム)
- **解説タイム**: ミス・時間切れ・パスのあとは3秒間、正解(漢字なら読み方と意味)の解説が表示される
- **ふりかえり**: ゲーム終了後、今回出題された問題の一覧(正誤・読み方・意味)を確認できる
- **演出・効果音**: バトル開始のREADY GO+ゴング、攻撃・命中・相殺・勝敗などの効果音(Web Audioで合成、音声ファイル不要)。ヘッダーの🔊ボタンでON/OFF
- **設定(入力スピード)**: ヘッダーの⚙️ボタンから「ゆっくり/ふつう/はやい」を選択可能。タイピング速度の個人差に合わせて、敵の攻撃間隔・サバイバルの制限時間・スピードボーナスの基準がまとめて調整される。localStorageに保存され次回訪問時も維持
- **プロフィール(準備中)**: ヘッダーの👤ボタンからプレイヤー名を設定可能(localStorageに保存)。称号・バッジはカタログと表示欄のみ用意した段階で、獲得条件はこれから実装
- **オンライン対戦(準備中)**: モード選択にカードは用意済みだが、実際の対戦はまだ未実装(押すと準備中の通知が出る)
- **難易度**: 漢字は漢検の級を目安にスライド(Lv1=10級〜8級 … Lv10=2級〜1級)。計算はモードによってカーブが異なる
  - スピードバトル・ステージ・ちょうどいいバトル(時間に追われるモード): 桁数がゆるやかに増加(加減算はLv10で4桁程度、乗除算は2桁中心でLv9〜10のみ3桁が少し混じる程度に抑制)
  - サバイバル(考える時間があるモード): Lv1-2:1桁 〜 Lv10:5桁と、より本格的な難易度カーブ

## 開発

ローカルで開発する場合:

```
# 任意の静的サーバーで配信(例)
npx serve .
```

もしくは `index.html` を直接ブラウザで開いてもOK。

# ケイサンバトル

計算・漢字の読み方・歴史などを解いて相手を攻撃する、ブラウザで動く対戦ゲーム(試作版)。
`index.html`(ゲーム本体)+ `kanji-data.js`(漢字語彙データ)+ `history-data.js`(歴史データ)の3ファイル構成。ビルド不要・依存ライブラリなし。

## 画面構成

**ホーム → 教科をえらぶ(けいさん/かんじ/れきし/…) → モードをえらぶ → レベルをえらぶ → バトル**、という階層構造。
設定・プロフィールもホーム画面のタイルから開ける(ヘッダーのアイコンからも同じ画面を開けるので、プレイ中でも変更可能)。

## 遊び方

`index.html` をブラウザで開くだけ。スマホでのプレイを想定した設計。

- **教科**: 「けいさん」「かんじの読み方」「れきし」から選択(かがく・えいご・おんがく・びじゅつは近日公開。今は選ぶと「準備中」通知が出る)
- **スピードバトル**: 問題を解いて攻撃。相手も自動攻撃してくる。先にHPを0にした方が勝ち
- **サバイバル**: 制限時間内に解き続け、5ミスで終了。スコア(正解数)を伸ばす
- **レベルモード**: Lv.1〜10のステージクリア型。クリアで次ステージが解放
- **診断バトル**: 全12問のスコアアタックで自分の実力を診断。正解が続くと問題の難易度が自動で上がり、間違えるとやさしく戻る。今ちょうどいいレベルを知りたいときにも使える
- **ミス問題**: これまでのバトルで間違えた問題だけを教科ごとに再挑戦できる復習モード。正解すると卒業してリストから消える。間違えた問題はlocalStorageに自動で蓄積される

### 特徴的な仕組み

- **攻撃演出**: 解答すると攻撃が約1秒かけて相手側へ飛び、着弾時にダメージ。双方の攻撃が同時に飛んでいると「相殺(クラッシュ)」が発生し、威力の差分だけが届く
- **正解・不正解エフェクト**: 解答すると画面中央に◎(正解)/✕(不正解)のエフェクトが表示される
- **入力**: 計算モードは画面上テンキー(スマホキーボードを出さない)、漢字・歴史モードは端末標準のひらがな入力。物理キーボードにも対応
- **パス**: 現在の問題をノーカウントでスキップ(3秒クールタイム)
- **解説タイム**: ミス・時間切れ・パスのあとは3秒間、正解(漢字・歴史なら読み方と意味)の解説が表示される
- **連続出題防止**: 直前と同じ問題(漢字・歴史の単語、計算式)が連続で出ないよう引き直す仕組み
- **ふりかえり**: ゲーム終了後、今回出題された問題の一覧(正誤・読み方・意味)を確認できる
- **演出・効果音**: バトル開始のREADY GO+ゴング、攻撃・命中・相殺・勝敗などの効果音(Web Audioで合成、音声ファイル不要)。ヘッダーの🔊ボタンでON/OFF
- **一時停止**: バトル中はヘッダーの⏸ボタンでいつでも一時停止できる。「つづける」で敵の攻撃間隔やサバイバルの残り時間を正確に引き継いで再開、「ホームに戻る」で対戦を切り上げられる
- **設定(入力スピード)**: ホーム画面の⚙️タイル(またはヘッダーの⚙️ボタン)から5段階(とてもゆっくり/ゆっくり/ふつう/はやい/とてもはやい)を選択可能。タイピング速度の個人差に合わせて、敵の攻撃間隔・サバイバルの制限時間・スピードボーナスの基準がまとめて調整される。選択中のボタンは色が変わって分かりやすく表示され、localStorageに保存され次回訪問時も維持
- **プロフィール(準備中)**: ホーム画面の👤タイルからプレイヤー名を設定可能(localStorageに保存)。称号・バッジはカタログと表示欄のみ用意した段階で、獲得条件はこれから実装
- **オンライン対戦(準備中)**: ホーム画面にタイルは用意済みだが、実際の対戦はまだ未実装(押すと準備中の通知が出る)
- **難易度**: 漢字は漢検の級、歴史は時代の進み方を目安にレベルをスライド。計算はモードによってカーブが異なる
  - スピードバトル・ステージ・診断バトル(時間に追われるモード): 桁数がゆるやかに増加(加減算はLv10で4桁程度、乗除算は2桁中心でLv9〜10のみ3桁が少し混じる程度に抑制)。敵の攻撃間隔も全体的にゆっくりめに調整
  - サバイバル(考える時間があるモード): Lv1-2:1桁 〜 Lv10:5桁と、より本格的な難易度カーブ
- **難読漢字**: 漢字モードのLv7以降には、意気地・固唾・十八番・杜撰・逼迫など読み方が難しい言葉を散りばめている

## 開発

ローカルで開発する場合:

```
# 任意の静的サーバーで配信(例)
npx serve .
```

もしくは `index.html` を直接ブラウザで開いてもOK。
